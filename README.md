## 更新

2025.08.24

- 添加支持在Docker环境安装的时候，选择是否安装为可进行硬盘大小限制，选择是否非默认路径安装，选择安装的存储池大小
- 检测btrfs存储时，开设容器添加可支持设置硬盘大小的选项
- 添加脚本可自行检测Docker是否真正限制了容器硬盘的大小
